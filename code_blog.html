Self Join Proof:

In this blog, I prove that if x represents a value in column y, x.row_count represents the number of rows with value x in column y, and j represents the number of self joins to column y, then the number of rows returned by the query is equal to the following: (x1.row_count)^j + (x2.row_count)^j + ... (xn.row_count)^j
